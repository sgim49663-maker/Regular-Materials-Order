# 🧪 실험실 물품 주문 관리 시스템

실험실에서 정기적으로 주문하는 물품의 **주문 주기**를 관리하고, **D-day**로 다음 주문 시점을 한눈에 확인할 수 있는 웹 앱입니다.

---

## 📌 주요 기능

### 1. D-day 확인
- 메인 화면에서 각 물품의 **다음 주문 예정일까지 남은 일수**를 D-day로 표시합니다.
- 긴급한 물품일수록 위에 표시됩니다.
  - 🔴 **빨간색**: 기한 초과 또는 7일 이내
  - 🟠 **주황색**: 14일 이내
  - 🟢 **초록색**: 여유 있음

### 2. D-day 기준 주기 선택
- 좌측 사이드바에서 **최소 주문주기** 또는 **평균 주문주기** 중 하나를 선택할 수 있습니다.
  - **최소 주문주기**: 가장 빨랐던 주문 간격 기준 (보수적)
  - **평균 주문주기**: 평균 주문 간격 기준

### 3. 업체 필터
- 사이드바에서 **전체 / 케이바이오 / 코람** 중 선택하여 업체별로 물품을 필터링할 수 있습니다.

### 4. 마지막 주문일 변경
- 물품 카드를 펼치면 **마지막 주문일**을 변경할 수 있습니다.
- 주문일을 변경하면 D-day가 자동으로 재계산되며, 주문 이력에 기록됩니다.

### 5. 주문 이력 조회
- 물품 카드 내 **"이력 조회"** 버튼을 누르면 해당 물품의 주문일 변경 기록을 확인할 수 있습니다.

### 6. 물품 숨기기 / 다시 표시
- 당분간 주문하지 않는 물품은 **"🙈 숨기기"** 버튼으로 목록에서 숨길 수 있습니다.
- 사이드바에서 **"🙈 숨긴 물품 보기"** 토글을 켜면 숨긴 물품을 확인하고, **"👁️ 다시 표시"** 버튼으로 복원할 수 있습니다.

### 7. 새 물품 추가
- 사이드바 하단의 **"➕ 새 물품 추가"** 폼에서 새로운 물품을 등록할 수 있습니다.
- 업체, Cat #, 품명, 주문 주기 정보 등을 입력합니다.

### 8. 물품 삭제
- 물품 카드 내 **"🗑️ 물품 삭제"** 버튼을 누르면 확인 후 삭제됩니다.
- 삭제된 물품은 복구할 수 없으니 주의해주세요.

---

## ⚠️ 주의사항

- **기존 스프레드시트와 연동되지 않습니다.** 이 앱은 별도의 데이터베이스를 사용하므로, 스프레드시트에서 수정한 내용이 이 앱에 반영되지 않고, 반대도 마찬가지입니다.
- **주문 주기(최소/평균)를 변경하고 싶은 경우** 앱에서 직접 수정이 불가합니다. 배포자에게 문의해주세요.
- 평균/최소 주기가 주문할 때마다 **자동으로 업데이트**되는 기능이 필요하시면 피드백 부탁드립니다. 추후 반영하겠습니다.
